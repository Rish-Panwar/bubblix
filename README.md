# 🥤 Bubblix – Modern Beverage Brand Experience

A visually rich, interactive product landing page for **Bubblix**, a probiotic soda brand. Built with modern web technologies and smooth animations to deliver a premium, engaging user experience.

---

## 🚀 Live Demo

👉 https://bubblix-six.vercel.app/

---

## ✨ Features

* 🎨 **Bold Typography Design** – Large, responsive, brand-focused text sections
* 🌀 **Smooth Animations** – Powered by GSAP for fluid transitions and interactions
* 🍊 **Interactive Flavor Carousel** – Animated product switching with dynamic backgrounds
* 💡 **Responsive Design** – Optimized for mobile, tablet, and desktop
* 🌊 **Custom Visual Effects** – Wavy backgrounds and floating elements for a playful feel
* ⚡ **High Performance** – Built with Next.js for fast loading and optimized rendering

---

## 🛠️ Tech Stack

* **Frontend:** Next.js (App Router)
* **Styling:** Tailwind CSS
* **Animations:** GSAP (GreenSock)
* **Image Optimization:** Next/Image
* **Deployment:** Vercel

---
